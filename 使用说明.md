## 人脸识别平台使用说明

### 一、安装依赖
- 使用npm安装依赖
```javaScript
npm install
```
 - 在安装yarn的情况下可以使用
```
yarn
```
### 二、运行
```
npm run start
```
or
```
yarn start
```
